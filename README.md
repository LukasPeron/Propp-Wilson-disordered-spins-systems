# Propp-Wilson Disordered Spins Systems

This repository contains the work I have done during my Master 2 Internship at INRIA and DI-ENS under the supervision of Dr. Louise Budzynski on an implementation of the Coupling From The Past Propp-Wilson algorithm for disordered spins systems.

All the results can be found in "figures" where the coupling time analysis are to be found in "algo" and the observables predictions are in "physics".
